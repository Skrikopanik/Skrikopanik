Ram-Minne

RAM eller Random Access Memory är ett minne där varje minnescell kan läsas eller skrivas till direkt utan att behöva läsa igenom andra delar av minnet, till skillnad från minnen som läses sekventiellt.

RAM används ofta som benämning på olika typer av läs- och skrivbart primärminnen, vilket kan vara förvirrande då RAM är benämningen på typen av åtkomst.
